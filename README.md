# GWAS Manhattan Plot Generator

Publication-quality GWAS visualisation toolkit.

## Features
- Manhattan plot with genome-wide significance line
- QQ plot with genomic inflation factor (lambda)
- Regional association plots (LocusZoom-style)
- Annotation of top hits
- Customisable colour schemes
- Export to PNG, PDF, SVG

## Usage
```bash
python gwas_plots.py --sumstats data/gwas_results.txt --output plots/
```
